Detailed Description

template<std::size_t dim>
class SASPSO< dim >

Specialization of the Optimizer class implementing the Self Adaptive Standard Particle Swarm Optimization (SASPSO) algorithm. All the information regarding the analysis and the theory behind this algorithm can be found in the following paper: http://dx.doi.org/10.1155/2016/8627083.

Parameters
swarm_sizethe number of particles in the swarm
max_iterthe maximum number of iterations
tolthe tolerance used for checking constraint conditions
omega_sthe starting value of the inertia weight
omega_fthe final value of the inertia weight
phi1_sthe starting value of the cognitive parameter
phi1_fthe final value of the cognitive parameter
phi2_sthe starting value of the social parameter
phi2_fthe final value of the social parameter
swarm_the std::vector storing the swarm of Particle_SASPSO instances
global_best_index_the index in the swarm array of the global best particle
violation_thresholdthe threshold used for considering a particle as feasible or not
Template Parameters
dimthe dimension of the space in which the function is defined
