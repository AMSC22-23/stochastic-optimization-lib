Detailed Description

template<std::size_t dim>
class ABC< dim >

Specialization of the Optimizer class implementing the Artificial Bee Colony Optimization (ABC) algorithm. All the information regarding the analysis and the theory behind this algorithm can be found in the following paper: https://link.springer.com/content/pdf/10.1007/978-3-540-72950-1_77.pdf.

Parameters
colony_sizethe number of particles in the swarm
max_iterthe maximum number of iterations
tolthe tolerance used for checking constraint conditions
Template Parameters
dimthe dimension of the space in which the function is defined
