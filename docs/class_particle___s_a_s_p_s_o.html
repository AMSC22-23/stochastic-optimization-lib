Detailed Description

template<size_t dim>
class Particle_SASPSO< dim >

Class that represents a single particle in the swarm.

Parameters
problem_a shared pointer to the problem to be optimized
random_generator_a shared pointer to the marsenne twister random number generator
position_the position of the particle in the search space
velocity_the velocity of the particle in the search space
best_position_the best position ever reached by the particle
best_value_the value of the fitness function in the global best position
best_constraint_violation_the total constraint violation in the global best position
omega_s_the starting value of the inertia weight
omega_f_the final value of the inertia weight
phi1_s_the starting value of the cognitive parameter
phi1_f_the final value of the cognitive parameter
phi2_s_the starting value of the social parameter
phi2_f_the final value of the social parameter
Template Parameters
dimthe dimension of the space in which the function is defined
